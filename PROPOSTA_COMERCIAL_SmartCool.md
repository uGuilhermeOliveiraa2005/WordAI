### 4.2 Especificações Técnicas Principais

#### Câmara Frigorífica CF-01 (Alimentos Congelados)

- Temperatura de projeto: **-25°C** (interna)
- Volume interno: **400m³** (aproximadamente 80 posições-pallet)
- Paredes: painel sanduíche 150mm PIR com densidade de 40kg/m³
- Piso: placa isotérmica com resistência elétrica anti-congelamento
- Portas: bipartida automática com cortina de ar para minimização de perdas
- Iluminação: LED fria embutida com sensor de presença

#### Sistema de Refrigeração

- Refrigerante: **R-449A** (GWP < 1.400 — compliance total com regulamentação ambiental)
- Compressor: Bitzer semi-hermético scroll, 15HP, inversor de velocidade variável (VFD)
- COP esperado: **≥ 3,2** (@ -10°C / +32°C)
- Desgelo: por gás quente com ciclo reverso automatizado
- Tuberias: cobre sanitário com isolamento de elastômero

#### Plataforma IoT — SmartCool Monitor

- Sensores: temperatura (x4), umidade relativa (x2), alarme de porta (x2), consumo elétrico (x2)
- Comunicação: 4G/WiFi com gateway local e redundância
- Plataforma: dashboard web responsivo + aplicativo iOS/Android com alertas push e e-mail
- Integração: API REST para conexão com ERP do cliente
- Retenção de dados: **3 anos** em nuvem com backup diário automatizado
